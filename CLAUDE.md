# ikura(割り勘・立替精算アプリ 公開版)

作業を始める前に **必ず [ROADMAP.md](ROADMAP.md) を読むこと**。
収益化方針・技術スタックの決定事項・フェーズ別タスク・仕様がすべてそこに集約されている。

ルール:
- 決定事項が変わったら ROADMAP.md を直接更新し、末尾の更新履歴に1行追記する
- タスクを完了したら該当チェックボックスを `[x]` にする
- 送金機能は法的理由により永久に実装禁止(詳細は ROADMAP.md §2)

## 開発コマンド

- `npm run dev` — 開発サーバー起動
- `npm run build` — 本番ビルド(出力: `dist`)

## 構成

Vue 3 + Vite の SPA。ページは `src/pages/`(セッション作成 → メンバー編集 → 支出登録 → 精算)、DB アクセスは `src/api/supabase.js`。
